# GPU-Performance-Glossary

本文主要翻译自 https://github.com/modal-labs/gpu-glossary 的内容，提取了其中关于 GPU Performace 的术语。

该部分涵盖了在使用 NSight Compute 调试 GPU Kernel 性能问题时可能遇到的所有术语。

1. 性能瓶颈 （Peformance Bottleneck）
2. Roofline 模型 （Roofline Model）
3. 计算瓶颈 （Compute-bound）
4. 带宽瓶颈 （Memory-bound）
5. 计算强度 （Arithmetic Intensity）
6. 额外开销 （Overhead）
7. 李特尔定律 （Little's Law）	
8. 内存带宽 （Memory Bandwidth）	
9. 计算带宽 （Arithmetic Bandwidth）	
10. 延迟隐藏 （Latency Hiding）
11. Warp 执行状态 （Warp Execution State）	
12. 活跃周期 （Active Cycle）
13. 占用率 （Occupancy）
14. 管线利用率 （Pipe Utilization）	
15. 峰值速率 （Peak Rate）
16. 指令发射效率 （Issue Efficiency）
17. SM 利用率 （Streaming Multiprocessor Utilization）
18. Warp 分支发散 （Warp Divergence）
19. 分支效率 （Branch Efficiency）
20. 合并访存 （Memory Coalescing）	
21. Bank 冲突（Bank Conflict）
22. 寄存器压力 （Register Pressure）
