# 内存带宽 （Memory Bandwidth）

内存带宽（Memory bandwidth）是指 在内存层级的不同级别之间传输数据的最大速率。

它表示以 字节每秒 为单位的理论最大数据传输吞吐量。它决定了 屋顶线模型（roofline model） 中“内存屋顶（memory roof）”的斜率。

在完整系统中存在多个内存带宽——每一层内存层级之间都有带宽

其中最重要的带宽是 GPU RAM 与流多处理器（Streaming Multiprocessors, SMs）的寄存器文件之间的带宽，因为大多数内核的工作集只能放入 GPU RAM，而无法放入内存层级的更高层。正因如此，这个带宽是 GPU 内核性能屋顶线建模中主要使用的带宽。

当代 GPU 的内存带宽以 TB/s（太字节每秒） 计。例如，B200 GPU 对其 HBM3e 内存的（双向）内存带宽为 8 TB/s。这远低于这些 GPU 中 Tensor Core 的算术带宽（Arithmetic bandwidth），导致屋脊点（ridge point）的算术强度增加。

下表列出了从 Ampere 到 Blackwell 流多处理器架构的 NVIDIA 数据中心 GPU 的代表性带宽数值：

| 系统（计算/内存）                     | 算术带宽 (TFLOPs/s) | 内存带宽 (TB/s) | 屋脊点 (FLOPs/byte) |
| ----------------------------- | --------------- | ----------- | ---------------- |
| A100 80GB SXM BF16 TC / HBM2e | 312             | 2           | 156              |
| H100 SXM BF16 TC / HBM3       | 989             | 3.35        | 295              |
| B200 BF16 TC / HBM3e          | 2250            | 8           | 281              |
| H100 SXM FP8 TC / HBM3        | 1979            | 3.35        | 592              |
| B200 FP8 TC / HBM3e           | 4500            | 8           | 562              |
| B200 FP4 TC / HBM3e           | 9000            | 8           | 1125             |
