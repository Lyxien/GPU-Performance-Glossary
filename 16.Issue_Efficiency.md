# 指令发射效率 （Issue Efficiency）

Issue efficiency（指令发射效率） 衡量的是 warp 调度器（warp scheduler）是否能让执行管线保持忙碌，也就是每个周期是否成功从可执行的 warp 中发射（issue）一条指令。

## 如果 Issue efficiency = 100%

表示：
- 每个周期（cycle）调度器都成功发射了一条指令
- 每个周期至少有一个 warp 处于“可执行”（eligible）状态
- GPU 的执行管线一直满负荷工作（即，没有因为等待数据/资源而停顿）

## 如果 Issue efficiency < 100%

这说明有一些周期发生了以下情况：
- 所有活跃 warp 都 被阻塞（stalled），原因可能是
    - 等待内存加载
    - 等待寄存器/执行单元资源
    - 指令依赖没有准备好
- 调度器找不到可执行 warp，调度器空转（idle）了，导致指令吞吐下降

## 一句话总结

通过 Issue efficiency 可以知道 GPU 的 warp 调度器有多忙。

如果调度器没有每周期都发射指令，说明 warp 在等待资源、数据或依赖，GPU 没吃满。
